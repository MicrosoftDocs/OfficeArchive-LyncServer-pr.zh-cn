_**主题上次修改时间：** 2016-05-25_

使用 Lync 混合部署，你可以在本地和 Lync Online 中的其他用户共享相同的域。 这可让你在不同地理位置或远程连接的用户向你的组织的用户提供 Lync 服务变得更容易。 你还可以利用 Lync 混合配置作为 Microsoft 365 的迁移路径。

本部分介绍 Lync Server 2013 的混合配置。
