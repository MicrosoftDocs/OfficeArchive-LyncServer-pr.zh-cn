_**主题上次修改时间：** 2013-02-21_

*网络区域路由* 定义一对网络区域之间的路线。 呼叫许可控制部署中的每对网络区域都需要网络区域路线。 这样部署中的每个网络区域便能够访问任何其他区域。 使用此部分中的过程查看、创建、修改或删除网络区域路由。
