# <a name="remove-a-legacy-authorized-host-in-lync-server-2013-optional"></a>在 Lync Server 2013 中删除旧版授权主机（可选）

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主题上次修改时间：** 2012-09-05_

当你将远程呼叫控制迁移到 Lync Server 部署时，必须在旧版部署中删除旧版授权主机项 (在 Lync Server) 中被称为 " *受信任的应用程序" 条目* 。 必须使用管理工具分别从这些部署中删除授权的主机条目。
